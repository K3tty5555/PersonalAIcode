# AI_PM 模板库

存放各类产品文档模板，供 AI_PM 技能使用或参考。

## 目录结构

```
templates/
├── README.md                      # 本文件
├── example-requirement.md         # 需求示例
├── prd-templates/                 # PRD 模板
│   └── (上传你的 PRD 模板)
├── user-story-templates/          # 用户故事模板
│   └── (上传你的用户故事模板)
├── analysis-templates/            # 需求分析模板
│   └── (上传你的分析模板)
└── research-templates/            # 竞品研究模板
    └── (上传你的研究模板)
```

## 使用方式

1. **直接上传模板文件**到对应目录
2. **文件名建议**：使用 `-template.md` 后缀，如 `mobile-app-prd-template.md`
3. AI_PM 技能可以读取这些模板作为参考

## 模板格式建议

每个模板文件建议包含：
- 适用场景说明
- 文档结构大纲
- 示例内容（可选）
